# Hokku Maker — Project Overview

**Hokku Maker** is a study project (Codecademy Backend Engineer career path) created to demonstrate fundamental **Node.js** techniques in a simple and approachable way.  
The initial goal was to build a minimal application that highlights:

- Basic Node.js structure  
- Asynchronous JavaScript patterns  
- Simple project organization  

As the project evolved, it was expanded with additional technologies to make it more practical and closer to real‑world development.

---

## Express.js Integration

To improve routing and request handling, the project now uses **Express.js**.  
This addition provides:

- Cleaner and more modular routing  
- A structured approach to handling HTTP requests  
- A lightweight foundation for future expansion  

Express.js transforms the project from a basic Node.js script into a small but functional web service.

---

## OpenAI API Usage

The project also integrates the **OpenAI API** to generate short English hokku poems based on user prompts collected through basic form.

This demonstrates:

- How to work with external APIs in Node.js  
- Handling asynchronous API calls  
- Returning generated content through an Express.js endpoint  

The OpenAI integration is intentionally simple to keep the focus on clarity and learning.

---

## Purpose of the Project

Hokku Maker is designed purely for educational purposes, with accent on backend.  
It aims to:

- Illustrate foundational Node.js concepts  
- Show how Express.js can structure a small service  
- Demonstrate basic usage of the OpenAI API  
- Serve as a clean, readable example for beginners  

This project is not intended for production use, but rather as a compact and understandable reference for anyone learning Node.js and related technologies.
